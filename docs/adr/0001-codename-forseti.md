# ADR-0001 — Codename: Forseti

- **Status:** Accepted
- **Date:** 2026-06-16

## Context

The H2 2026 investment needed a program codename to brand the repo, the GitHub project, and
the writeups. "Zeus" was considered but felt too "supreme" — and is being deliberately
reserved for something bigger later. The deck's mascot is a **raven** (echoing Odin's ravens),
and the technical thesis is *establishing truth about code via proof / judgment*.

## Decision

The program is codenamed **Forseti** — the Norse god of justice and reconciliation who
presides over every dispute in his hall Glitnir and hands down the fairest judgment.

The working dir was renamed `dev/noname` → `dev/forseti` (2026-06-16); the public GitHub repo is
`pmatos/forseti`.

## Consequences

- The name maps cleanly onto the system's job: judge code, return a verdict (VERIFIED, or a
  concrete counterexample). Stays in the same Norse world as the raven mascot.
- "Glitnir" is available as a sub-name (e.g. dashboard or property store) if wanted.
- "Zeus" stays unused, reserved for a future, larger effort.
